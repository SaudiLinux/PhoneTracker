الوصف:
-------
PhoneTracker هي أداة شاملة ومتقدمة للبحث عن معلومات أرقام الهواتف
وجمع البيانات المرتبطة بها من مصادر متعددة على الإنترنت.

الميزات الرئيسية:
-----------------
✓ التحقق من صحة أرقام الهواتف وتنسيقها
✓ الحصول على المعلومات الأساسية (الموقع، شركة الاتصالات)
✓ البحث في Truecaller للحصول على الأسماء
✓ البحث في مواقع التواصل الاجتماعي:
  - Facebook
  - Instagram  
  - Twitter
  - LinkedIn
  - Telegram
  - WhatsApp
✓ البحث عن الإيميلات المرتبطة بالرقم
✓ البحث المتقدم في محركات البحث
✓ البحث في قواعد البيانات المسربة
✓ البحث في المنتديات والمواقع
✓ تجاوز جدران الحماية الأساسية
✓ إنشاء تقارير مفصلة بتنسيق JSON

المتطلبات:
----------
1. Python 3.7 أو أحدث
2. نظام التشغيل: Windows/Linux/macOS
3. اتصال مستقر بالإنترنت
4. متصفح Google Chrome (للبحث المتقدم)
5. ذاكرة RAM: 4GB على الأقل
6. مساحة تخزين: 500MB على الأقل

التثبيت:
--------
1. تأكد من تثبيت Python على نظامك
2. قم بتشغيل ملف install_requirements.bat:
   - في Windows: انقر نقراً مزدوجاً على الملف
   - أو من سطر الأوامر: install_requirements.bat

3. أو قم بالتثبيت اليدوي:
   pip install -r requirements.txt

4. تأكد من تثبيت Google Chrome

طريقة الاستخدام:
---------------

الاستخدام الأساسي:
python phonetracker.py [رقم_الهاتف]

أمثلة:
------
# البحث عن رقم سعودي
python phonetracker.py +966501234567

# البحث عن رقم مصري
python phonetracker.py +201012345678

# البحث عن رقم أمريكي
python phonetracker.py +15551234567

# البحث مع حفظ النتائج في ملف محدد
python phonetracker.py +966501234567 -o my_report.json

# البحث مع عرض تفاصيل إضافية
python phonetracker.py +966501234567 -v

# البحث بدون مواقع التواصل الاجتماعي
python phonetracker.py +966501234567 --no-social

الخيارات المتاحة:
----------------
-h, --help          عرض رسالة المساعدة
-o, --output        تحديد ملف الإخراج للتقرير
-v, --verbose       عرض تفاصيل إضافية أثناء البحث
--no-social         تخطي البحث في مواقع التواصل الاجتماعي

استكشاف الأخطاء:
-----------------

1. خطأ "Module not found":
   الحل: pip install -r requirements.txt

2. خطأ "Invalid phone number":
   الحل: تأكد من تنسيق الرقم الصحيح (+966501234567)

3. خطأ "Connection timeout":
   الحل: تحقق من اتصال الإنترنت واستخدم VPN

4. خطأ "Chrome driver not found":
   الحل: تأكد من تثبيت Google Chrome

5. خطأ "Permission denied":
   الحل: شغل الأداة بصلاحيات المدير

6. خطأ "Rate limit exceeded":
   الحل: انتظر قليلاً ثم أعد المحاولة

7. خطأ "SSL Certificate":
   الحل: استخدم --no-ssl-verify
